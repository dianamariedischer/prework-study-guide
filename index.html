HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element is another section of visible content, but at the bottom of the webpage.
  • The meta element can be used for:
    • SEO
    • Setting a character encoding system (UTF-8 is common)
    • Viewport settings
    • And like other stuff haha
  • HTML creates the structure of the webpage.

Git

Commands:

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: takes you back to the main branch
  • git pull origin branch-name: not really sure
  • git push origin branch-name: i think this pushes a branch up for review in github?
  • git add -A: adds all unstaged files to staging
  • git commit -m "decription of adjustments": commits staged files with description
